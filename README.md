# LessonTrack

A web-based Lesson Planning and Academic Milestone Tracking System for college instructors.

## Features

- Subject Management
- Lesson Management
- Quiz and Activity Tracking
- Academic Milestone Monitoring
- Dashboard Analytics

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap
- Maven
